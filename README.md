# WeatherNOW: Reporter LIVE

A game where you are a meteorologist, attempting to predict the weather for the next day. Each day, a weather system roles with events, the player is a weatherforcaster trying to predict tommorows weather based on todays patterns. the closer your guess, the higher you score.

## How to Run

### In a terminal

1. Install dependencies

```bash
   pip install asciimatics blessed pyfiglet
```

2. Run the program.

```bash
   python main.py
```
